ChronoCanvas API

Una API robusta para la gestión de registros históricos de la Iglesia de Jesucristo de los Santos de los Últimos Días

Registros Históricos

Accede y gestiona registros históricos importantes de la Iglesia, con soporte para fechas, descripciones y referencias.

Dedicaciones de Templos

Consulta información detallada sobre las dedicaciones de templos, incluyendo fechas y autoridades oficiantes.

Autenticación Segura

Sistema de autenticación robusto con soporte para JWT y OAuth 2.0 a través de GitHub.

Endpoints Principales

  • GET /ldsChurchHistory - Obtener registros históricos
  • GET /templeDedications - Consultar dedicaciones de templos
  • POST /auth/register - Registro de usuarios
  • POST /auth/login - Inicio de sesión
